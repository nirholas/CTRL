This is the website link for LyraOS for larger-screen devices.

To use LyraOS in a mobile or a vertically screened device with a dynamic viewport, consider installing the OS as an app.

## Installing LyraOS as an App in Chrome

Chrome, or its alternatives like Edge, Opera, have mostly the same methods.

{% stepper %}
{% step %}
### Open Chrome

From your default OS.
{% endstep %}

{% step %}
### Go to [lyra.surf](https://lyra.surf)

and wait until LyraOS installs.

Once it's done, and you see a dock,
{% endstep %}

{% step %}
### Tap the menu button

on Chrome; you can find it in the top right corner outside LyraOS.
{% endstep %}

{% step %}
### Tap "Install LyraOS" or "Add to home screen"

Now wait for your OS to make a dialog for its installation, install LyraOS, and you are done.
{% endstep %}

{% step %}
### From the app menu on your device, open LyraOS.

You now have your personal Web System ready to be used.
{% endstep %}
{% endstepper %}
